News HUB

The application contains news from a large number of american and british news websites. These are obtained using News API. On the main page there is an overview of the most popular and the latest news. User may create an account that will enable personalization of displayed content. The application allows logged users to choose up to 7 categories offered by News API, select visibility of chosen ones and search for information.

Account data is stored by using a NoSQL database provided by Google - Firebase. Logging in to the application is possible thanks to the traditional method (email, password) and through the account on the social networking site Facebook, or Google account.

The news on the registered user's home page is customized to his preferences. User will decide which categories will be displayed on the categories menu. On the side screen are available information about the weather in the city chosen by application developers - Krakow. This functionality is possible thanks to the use of Open Weather Map API.
